The following files and folders are in the directory:

Folder DATA contains two SAS Data files used by example programs.
Folder IML contains the power program POWERLIB203.IML
Folder EXAMPLES contains example programs that use POWERLIB203.IML.
  Both .SAS and .LST files are included, as well as .CGM  and .EMF graphic files 
  produced by a few of the programs.
  The manual and the README2.txt file within the EXAMPLES folder
  give more details about the examples.

Folder TEXT contains the manual and associated files.
  MANUAL203.PDF contains the manual for POWERLIB203.IML, in PDF format.
  It can be read and printed with many programs, including
  Adobe Acrobat Reader(r), which can be downloaded from the Adobe site at no cost.

  The file MANUAL203.WXP is an EXP(r) file, which requires the style sheet BOOK05.ESS.
